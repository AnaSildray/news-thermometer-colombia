# Contribuyendo a News Thermometer Colombia

¡Gracias por tu interés en contribuir! Aquí están las directrices:

## 📋 Cómo Contribuir

1. **Fork** el repositorio
2. **Crea una rama** para tu feature (`git checkout -b feature/AmazingFeature`)
3. **Commit** tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. **Push** a la rama (`git push origin feature/AmazingFeature`)
5. **Abre un Pull Request**

## 🐛 Reportar Bugs

- Abre un issue con descripción clara
- Incluye pasos para reproducir
- Incluye screenshots si aplica

## 💡 Sugerir Features

- Abre un issue con la etiqueta `enhancement`
- Describe claramente qué quieres lograr
- Explica por qué sería útil

## 📝 Estándares de Código

- Python: PEP 8
- JavaScript: ESLint
- Usa type hints en Python
- Documenta tus funciones

## 🧪 Testing

Antes de hacer PR:
```bash
# Backend
pytest backend/
flake8 backend/

# Frontend
npm test
npm run build
```
